# MDZ_Slider 幻灯片制作工具

### 软件工程小组名单：马天野，杜文豪，王致博，马润豪，张翔宇，吴冠霖，刘滨 （仅供作业认证，该签名将于2025年5月2号之后删除）

一个基于JavaFX的跨平台幻灯片编辑与展示工具，具有现代化界面和丰富的编辑功能。

## 功能特点

### 幻灯片管理
- 创建、切换、预览幻灯片
- 支持幻灯片序列化保存与加载
- 简洁的幻灯片导航控制

### 文本编辑
- 添加可拖拽定位的文本元素
- 支持字体大小、颜色、粗体、斜体样式调整
- 文本实时大小调整与控制点交互

### 图片处理
- 支持多种图片格式(PNG/JPG/JPEG/GIF/BMP)
- 智能缩放与边界控制
- 八向控制点精确调整图片大小

### 绘图功能
- 支持矩形、圆形、直线、箭头等基本形状绘制
- 线宽与颜色自定义
- 绘图元素选择与调整

### 交互设计
- 拖拽移动与调整所有元素
- 直观的选择反馈与控制点显示
- 右键菜单快速操作
- 键盘快捷键支持

### 放映功能
- 全屏放映模式
- 简洁的切换控制
- 流畅的演示体验

## 技术架构

### MVC架构设计
- **模型层**：Slide和各种SlideElement实现
- **视图层**：JavaFX画布与控件组合
- **控制层**：事件处理与用户交互逻辑

### 核心类结构
- `SlideElement`：所有幻灯片元素的抽象基类
  - `TextElement`：文本元素实现
  - `ImageElement`：图片元素实现
  - `DrawElement`：绘图元素实现
- `Slide`：幻灯片模型，管理元素集合
- `Main`：主应用入口与UI控制
- `PresentationWindow`：演示模式实现

### 设计模式应用
- **抽象工厂**：元素创建与管理
- **观察者模式**：UI状态更新
- **命令模式**：用户操作处理
- **序列化模式**：文件保存与加载

## 技术实现细节

### Java与JavaFX技术
- 基于Java 17开发
- JavaFX 21.0.1作为GUI框架
- Canvas绘图系统实现元素渲染
- CSS样式定制美观界面

### 元素渲染与交互
- 基于GraphicsContext的绘图操作
- 元素选择与拖拽算法
- 控制点计算与大小调整逻辑

### 序列化与持久化
- 基于Gson的JSON序列化
- 自定义序列化器处理复杂对象
- 文件读写操作封装

### 界面设计
- 现代化渐变主题
- 响应式按钮与控件
- 流畅的动效与交互反馈

## 系统要求

- **操作系统**：Windows, macOS, Linux等支持Java的系统
- **Java版本**：Java 17或更高版本
- **内存要求**：至少4GB RAM
- **存储空间**：100MB可用空间

## 构建与运行

1. 安装Java 17或更高版本
2. 下载JavaFX SDK 21.0.1

```bash
# 创建lib目录并下载JavaFX
mkdir -p lib
cd lib
curl -L -o javafx-sdk.zip https://download2.gluonhq.com/openjfx/21.0.1/openjfx-21.0.1_osx-aarch64_bin-sdk.zip
unzip javafx-sdk.zip
mv javafx-sdk-21.0.1 javafx-sdk-21.0.1
cd ..
```

3. 使用Gradle构建和运行

```bash
# 构建
./gradlew clean build

# 运行
./gradlew run
```

## 使用指南

### 基本操作
1. **创建幻灯片**：点击"新建幻灯片"按钮
2. **切换幻灯片**：使用"上一页"/"下一页"按钮或键盘方向键
3. **保存/打开**：通过文件菜单操作

### 添加内容
- **文本**：点击"添加文本"按钮，输入文字内容
- **图片**：点击"添加图片"按钮，选择本地图片文件
- **绘图**：选择绘图工具按钮，在画布上拖拽绘制

### 元素编辑
- **选择**：鼠标单击元素
- **移动**：拖拽已选中元素
- **调整**：拖拽选中元素的控制点
- **样式**：使用工具栏中的样式控件调整
- **删除**：按退格键或使用右键菜单删除

### 演示模式
- **启动**：通过"放映"菜单启动全屏演示
- **操作**：使用方向键或空格键切换幻灯片
- **退出**：按ESC键退出演示模式
